# .NET examples

This folder contains .NET/C# examples that use DecentDB in embedded mode. Both file-based and in-memory (`:memory:`) variants are included.

- [dapper-basic/](dapper-basic/) — Dapper with file-based database
- [dapper-memory/](dapper-memory/) — Dapper with `:memory:` in-memory database
- [microorm-linq/](microorm-linq/) — Micro-ORM (DbSet + LINQ) with file-based database
- [microorm-memory/](microorm-memory/) — Micro-ORM with `:memory:` in-memory database
- [entityframework/](entityframework/) — Entity Framework Core provider with `:memory:` database
