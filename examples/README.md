# Examples

This directory contains small, focused example projects that show how to use DecentDB from other languages and ecosystems.

- [dotnet/](dotnet/) — C#/.NET examples (Dapper + Micro-ORM)
- [go/](go/) — Go example using `database/sql` interface
- [node/](node/) — Node.js example using the N-API native addon
- [python/](python/) — Python example using DB-API 2.0 interface
- [showcase.sh](showcase.sh) — Quick CLI walkthrough of core DecentDB features
